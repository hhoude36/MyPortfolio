Eazy AZ 123 is an online version of the live Price is Right Game. The user is asked to rank the items from least to most expensive. If they get the order right, they win a prize.

Eazy AZ 123 is an online version of the live Price is Right Game. The user is asked to rank the items from least to most expensive. If they get the order right, they win a prize.

Role

This project was developed independently. Front-end design and development using CSS, HTML, and JavaScript.

Goals

My goal was to keep my JavaScript skills sharp and to learn how to implement the drag and drop function.

Design

I had a lot of fun modeling the design off of the live set. I used this screenshot as a guide as I chose the colors and fonts. I enjoyed making choices that might not have occurred to me otherwise.

Challenges

With a lot of interaction in just one small page, I learned a lot about JavaScript and am excited about my new skills. Games have proven to be a great way to practice what I know and add new tools to my toolkit. One challenge I faced was checking the user's answer against the correct answer, and extracting the correct information from the drag and dropped blocks.
