Stitch 'N Bitch

Visit Site

Inspired by the post WWII phenomenon known as the Stitch ‘N Bitch, crafters can find and build groups that meet regularly to quilt, knit, crochet, needlepoint, sew, and do other handicrafts. Additionally, they can track their own and others’ progress on current and past projects. Historically, women have used these gatherings as a way to gather and build community. During the 80s and 90s women in the Feminist, Punk, and DIY circles brought these craft circles back to life, using them as a place to reclaim women's work and build political movements. Ever since, the Stitch 'N Bitch has remained alive and well but there has never been a platform dedicated entirely for them. With Stitch 'N Bitch it's never been easier to build community around the activities you love.

Role

This project was developed collaboratively withBetty-Lee Carter. We shared the responsibilities of ideation, front end, back end, and design. I focused on the functionalities related to searching for, participating in, and creating crafting groups.

Goals

To build a user-friendly site while solidifying my understanding of REACT, MUI, and PostgreSQL to build an API and consume it from the front end.

Design

The design aims to bridge across generations. While these traditional handicrafts are often associated with older crowds, we also wanted to keep in mind the resurgence among young, hip crafters. We referenced punk/DIY/feminist aesthetics while choosing our fonts and color scheme, while also ensuring the site was desktop friendly and fully mobile.

Challenges

I learned a ton while making this project. This was the first collaborative project I had worked on, and it was great to get comfy with Git and GitHub along with ElephantSQL to better share our API. We also uploaded images to our back end using Cloudinary, a skill I'm pleased to now have in my back pocket.

Future Plans

This project is a work-in-progress. There is a lot of functionality that we plan to add. Ultimately we want to lean into the social aspect, increingse the interaction between users and allowing for them to chat with one another and react to each other's projects. We also want to allow the option for a group to be invite only, allowing for a more exclusive experience.
